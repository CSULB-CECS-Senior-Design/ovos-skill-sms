# OVOS-SKILL-SMSM (OVOSSkill)

## About
OVOS will text one phone number a predifined text when asked. 
### Demo
https://youtu.be/Me-3pXEmSA8

## Examples Phrases
- "text 9 1 1"
- "text for help"
- "contact help"
- "contact emergency responders"
- "get help fast"
